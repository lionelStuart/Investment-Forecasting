# Investment Agent Output Contract

Return JSON only. Include `status`, `role`, `role_key`, `outcome`, `summary`,
`reason`, `analysis`, `reflection`, `risk_note`, and `evidence_ids` when the
schema requires them.

Use research-only wording. Avoid guaranteed-return, capital-protection, or
certainty language.
