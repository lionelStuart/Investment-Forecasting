# Investment Model Evidence Skill

Use stored model predictions, backtests, validation metrics, reliability
sidecars, and monitoring reports as evidence. Candidate models are contextual
unless promoted by system governance.

Downgrade stale, degraded, low-confidence, negative Rank IC, weak bucket-spread,
or outlier model evidence to watch-only context. Cite prediction, backtest, or
monitoring evidence IDs for any model claim.
